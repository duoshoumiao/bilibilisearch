# bilibilisearch
B站视频搜索

使用方法：
1. 查视频 [关键词] - 搜索B站视频
2. 视频关注 + 视频链接  -自动关注up 
3. 取关up [UP主名称] - 取消监控
4. 查看关注 - 查看当前监控列表


2025/07/13 改进推送效率，只能关注 查up 功能找到的UP，希望有大佬帮忙优化以下
